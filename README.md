# 🚀 GitHub Repo Analyser

**GitHub Repo Analyser** is an intelligent AI-powered tool that analyzes your GitHub repositories to generate detailed **work reports** and **task reports** — based on real code changes, not just commit messages.  
It leverages **Large Language Models (LLMs)** to deeply understand code modifications, summarize developer activity, and provide actionable insights for project tracking.

---

## 🧠 Features

- **LLM-Powered Code Understanding**  
  Understands actual code diffs instead of relying on commit messages, generating human-readable summaries.

- **Smart Work & Task Reports**  
  Automatically classifies changes into logical work items and tasks.

- **Language-Aware Analysis**  
  Detects meaningful structural changes (additions, deletions, refactors) across multiple programming languages.

- **Developer Insights**  
  Tracks contributions, productivity, and module-level ownership patterns.

- **Customizable AI Providers**  
  Supports multiple AI backends like Gemini, OpenRouter, and Ollama, so you can choose your preferred LLM.

---

## ⚙️ LLM Configuration

The analyser allows you to configure LLM settings directly through the interface.

| Field | Description |
|-------|--------------|
| **Provider** | Select your preferred LLM provider (`Gemini`, `OpenRouter`, or `Ollama`) |
| **Model** | Choose a model from the **dropdown list** corresponding to the selected provider |
| **API Key** | Enter your API key for the chosen provider |
